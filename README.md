ForoHub is a RESTful API built with Spring Boot that manages a discussion forum. It allows users to authenticate, create, modify, delete, and view forum topics, ensuring data integrity with validations and secure authentication using JWT. The API also supports pagination for fetching topics.

## Features

- **User Authentication**: Users can authenticate using JWT (JSON Web Token).
- **Topic Management**: Users can create, modify, delete, and retrieve forum topics.
- **Data Validation**: Ensures topics are not created with duplicate titles or messages.
- **Pagination**: Paginated topic results for better navigation and handling of large datasets.
- **Error Handling**: The API handles errors and exceptions, returning appropriate HTTP status codes and informative error messages.
- **Secure Architecture**: User authentication and authorization are securely managed with Spring Security and JWT.

## Technologies Used

- **Java 17**
- **Spring Boot**
- **Spring Security**: For user authentication and security management.
- **JPA (Java Persistence API)**: For database interaction.
- **JWT (JSON Web Tokens)**: For token-based authentication.
- **BCrypt**: For password encoding.
- **Swagger**: For API documentation.

## Endpoints

### Authentication

- **POST /login**: Allows users to authenticate with their username and password. Returns a JWT token.

  **Request Body:**
  
  {
    "user": "username",
    "pass": "password"
  }
  

  **Response:**
  
  {
    "jwtToken": "jwt_token_generated"
  }


### Topic Management

- **POST /topics**: Creates a new forum topic.

  **Request Body:**
  
  {
    "title": "Topic title",
    "message": "Topic message",
    "author": "Author name",
    "course": "Course name"
  }
  

  **Response:**
  
  {
    "title": "Topic title",
    "message": "Topic message",
    "author": "Author name",
    "course": "Course name"
  }
  

- **GET /topics/{id}**: Retrieves details of a topic by its ID.

  **Response:**
  
  {
    "title": "Topic title",
    "message": "Topic message",
    "date": "Creation date",
    "status": true,
    "author": "Author name",
    "course": "Course name"
  }
  

- **GET /topics**: Retrieves a paginated list of all topics.

  **Response (Paginated):**
  
  {
    "content": [
      {
        "title": "Topic title",
        "message": "Topic message",
        "date": "Creation date",
        "status": true,
        "author": "Author name",
        "course": "Course name"
      }
    ],
    "totalPages": 1,
    "totalElements": 1
  }
  

- **PUT /topics/{id}**: Updates an existing topic by ID.

  **Request Body:**
  
  {
    "title": "New title",
    "message": "New message",
    "author": "New author",
    "course": "New course"
  }
  

  **Response:**
  
  {
    "title": "New title",
    "message": "New message",
    "author": "New author",
    "course": "New course"
  }
  

- **DELETE /topics/{id}**: Deletes a topic by ID.

  **Response:**
  
  {}
